# Recipe-webpage-exercise
Make a webpage of recipe with HTML and CSS.
## 要求：
<p>制作一个食谱页面（HomePage），要求首页显示食物名与图片（2-3种），点击食物名字后跳转到对应的别的网页，跳转到的网页有食物的基本介绍与图片、制作过程、返回HomePage的按钮（button）</p>
<p>因此其实你最少需要做3个webPage，希望你将文件组织好，例如新建一个img文件夹专门存放图片，新建recipes文件夹存放每一个食物对应的页面，新建css文件夹存放所有样式等等</p>
<p>发挥你的创造力与审美能力，设计一个好看的页面吧！该project有充分的自由度，因为只给了一个html框架（</p>
